## Common Parts

Open infrastructure for the distribution of digital spare parts.

Common Parts builds the tools and protocols that make the Right to Repair work at the technical level — starting with a structured, open index of 3D-printable spare parts, and working toward a standardized protocol for authorized digital manufacturing.

---

### Contributing

The index grows through curation and direct contributions. If you maintain open-licensed repair models and want them properly indexed with attribution, see [`CONTRIBUTING.md`](https://github.com/commonparts/common-parts-access/blob/main/CONTRIBUTING.md).

Code contributions follow the same file.
