## Common Parts

Open infrastructure for the distribution of digital spare parts.

Common Parts builds the tools and protocols that make the Right to Repair work at the technical level — starting with a structured, open index of 3D-printable spare parts, and working toward a standardized protocol for authorized digital manufacturing.

---

### What this is

**Common Parts Access** is an open index of digital spare parts — 3D-printable models curated from open-source sources (Printables, Thingiverse, GitHub) and contributed directly. Every entry carries structured metadata, original attribution, and a license.

**The CPSP protocol** (Common Parts Secure Print) is a technical specification for authorized digital manufacturing — enabling a manufacturer to distribute a part file without losing control over how it's used. The spec is in development. It will be published as CC-BY when it reaches v1.0.

This is not a marketplace. Common Parts does not sell files.

### Contributing

The index grows through curation and direct contributions. If you maintain open-licensed repair models and want them properly indexed with attribution, see [`CONTRIBUTING.md`](https://github.com/commonparts/common-parts-access/blob/main/CONTRIBUTING.md).

Code contributions follow the same file. Open an issue before starting significant work.
